## 🎯 Beschrijving

Een webapplicatie die een volledig overzicht biedt van alle Valorant agents, hun rollen, abilities en meer. Gebouwd met Node.js, Express en MongoDB.

---

## ⚡ Features

| Feature | Beschrijving |
|---|---|
| 🔍 **Zoeken & Filteren** | Filter agents op rol en zoek op naam |
| ↕️ **Sorteren** | Sorteer op naam of releasedatum |
| 🧑‍💼 **Agent Detail** | Bekijk abilities met video's per agent |
| 🎭 **Rollen** | Overzicht van alle rollen met beschrijving |
| 👥 **Gebruikersbeheer** | Admins kunnen gebruikers verwijderen |
| ✏️ **Agent bewerken** | Admins kunnen agents aanpassen |
| 🔐 **Authenticatie** | Inloggen en registreren met sessies |

---

## 🛠️ Technologieën

- **Node.js** — runtime
- **Express.js** — webframework
- **TypeScript** — programmeertaal
- **MongoDB Atlas** — database
- **EJS** — templating engine
- **bcrypt** — wachtwoord hashing
- **express-session** — sessiebeheer
